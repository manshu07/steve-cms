# Metrics and KPIs

## Metric Categories

### North Star Metric
The single metric that best captures core value delivered to users.
- Examples: Weekly active users, Revenue per user, Time saved

### Business Metrics
- Revenue (MRR, ARR)
- Customer Acquisition Cost (CAC)
- Lifetime Value (LTV)
- Churn rate
- Net Promoter Score (NPS)

### Product Metrics
- Daily Active Users (DAU)
- Monthly Active Users (MAU)
- Session duration
- Feature adoption rate
- User retention

### Engineering Metrics
- Deployment frequency
- Lead time
- Mean time to recovery (MTTR)
- Bug escape rate
- Code coverage

## KPI Framework

### SMART Goals
- **S**pecific: Clear definition
- **M**easurable: Quantifiable
- **A**chievable: Realistic
- **R**elevant: Aligned with goals
- **T**ime-bound: Has deadline

### Metric Template
```markdown
## Metric: [Name]

| Attribute | Value |
|-----------|-------|
| Definition | [How it's calculated] |
| Data Source | [Where data comes from] |
| Target | [Goal value] |
| Current | [Current value] |
| Owner | [Responsible person] |
| Review Cadence | [How often reviewed] |
```

## Key Metrics by Product Type

### SaaS Products
| Metric | Description | Target |
|--------|-------------|--------|
| MRR | Monthly recurring revenue | Growing |
| Churn | % users leaving | <5%/month |
| NRR | Net revenue retention | >100% |
| CAC:LTV | Acquisition efficiency | 1:3 |

### E-commerce
| Metric | Description | Target |
|--------|-------------|--------|
| Conversion Rate | Visitors to buyers | >2% |
| AOV | Average order value | Growing |
| Repeat Rate | Returning customers | >30% |

### Mobile Apps
| Metric | Description | Target |
|--------|-------------|--------|
| DAU/MAU | Engagement ratio | >20% |
| D1 Retention | Day 1 return rate | >40% |
| D7 Retention | Day 7 return rate | >20% |

## Metric Reviews

### Weekly Review
- Check progress vs targets
- Identify anomalies
- Adjust tactics

### Monthly Review
- Trend analysis
- Goal adjustment
- Stakeholder updates

### Quarterly Review
- Strategic alignment
- Metric relevance check
- New metric definition
